Detailed Description

template<class MarginFunction>
struct dismec::objective::GenericMarginClassifier< MarginFunction >

A utility class template that, when instatiated with a MarginFunction, produces the corresponding linear classifier loss.

Template Parameters
MarginFunctionA class that needs to provide three scalar functions: value, grad, and quad that give the value, first order, and second order approximation to the function at the given point.
